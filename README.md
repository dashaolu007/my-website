# Medivo Health Technology Website

This is a refreshed, futuristic Medivo website package positioned as a general health technology company website, not a project-specific MVP website.

## Pages

- `index.html`: Futuristic Medivo homepage
- `platform.html`: Medivo platform overview
- `solutions.html`: Solutions for providers, patients, clinics, operations and partners
- `innovation.html`: AI, automation and interoperability vision
- `company.html`: Company mission and brand promise
- `contact.html`: Partnership and inquiry form
- `privacy.html`: Placeholder privacy page
- `terms.html`: Placeholder terms page

## Assets

- `assets/img/medivo-logo-main.png`: Uploaded Medivo logo used across the website
- `assets/img/favicon.png`: Generated favicon
- `assets/css/styles.css`: Futuristic responsive styling
- `assets/js/main.js`: Mobile navigation and demo form behavior

## Preview

Open `index.html` in a browser.

## Deployment

Deploy the entire folder to any static host such as Vercel, Netlify, Azure Static Web Apps, GitHub Pages, or a standard cPanel/hosting server.

## Before Public Launch

- Connect contact form to a secure backend, CRM, or email service.
- Replace placeholder privacy and terms pages with legally reviewed versions.
- Add final Medivo corporate contact details.
- Review all claims before external publishing.
