# Medivo Website Package

This is a static website package for Medivo, an Alberta-first EMR and digital health platform.

## Files

- `index.html`: Homepage
- `platform.html`: Product and feature overview
- `security.html`: Security and compliance page
- `investors.html`: Investor overview
- `contact.html`: Demo/contact form page
- `assets/css/styles.css`: Website styling
- `assets/js/main.js`: Mobile menu and static form behavior
- `assets/img/medivo-logo.svg`: Editable vector logo
- `assets/img/favicon.svg`: Browser icon

## How to view locally

Open `index.html` in a browser.

## How to deploy

You can deploy this folder to Vercel, Netlify, GitHub Pages, Azure Static Web Apps, or any standard web hosting provider.

## Important pre-launch items

- Replace placeholder form behavior with a secure backend or CRM integration.
- Add a formal Privacy Policy and Terms of Use.
- Add final corporate details for Medivo Technologies Inc.
- Review all healthcare, privacy, regulatory, and investment-related statements with counsel before public launch.
